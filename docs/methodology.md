# Financial Methodology

## Valuation Approaches
1. **Direct Capitalization**: NOI / Cap Rate.
2. **DCF**: PV of cash flows + terminal value.
3. **Sales Comparison**: GRM or PPU adjusted for phase/occupancy.

## Forecasting
- Revenue: Base * (1 + rent_growth)^year.
- Expenses: Base * (1 + expense_growth)^year.
- NOI: EGI - expenses - CapEx.

## Risk Assessment
- RAG thresholds based on industry standards.
- IRR sensitivity for scenario planning.
